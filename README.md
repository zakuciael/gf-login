# @zakku/gf-login

A node.js library for authenticating to GameFail's servers.

## Installation

#### NPM

```bash
npm install @zakku/gf-login
```

#### Yarn

```bash
yarn add @zakku/gf-login
```

#### PNPM

```bash
pnpm add @zakku/gf-login
```
